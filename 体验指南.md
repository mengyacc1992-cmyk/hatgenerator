# 🎩 帽子美学选题生成器 - 体验指南

## 📋 前置要求

- Node.js (v18 或更高版本)
- npm 或 yarn
- Gemini API Key（从 Google AI Studio 获取）

## 🚀 快速开始（3步体验）

### 步骤 1: 安装依赖

**安装前端依赖：**
```bash
npm install
```

**安装后端依赖：**
```bash
cd server
npm install
cd ..
```

### 步骤 2: 配置环境变量

创建后端环境变量文件：
```bash
cd server
cp .env.example .env
```

编辑 `server/.env` 文件，设置你的 Gemini API Key：
```bash
GEMINI_API_KEY=AIzaSyDLlYQiiLFKHHISU6Dy9fMA96qknEywlOc
PORT=3001
CORS_ORIGIN=http://localhost:3000
```

**如何获取 Gemini API Key：**
1. 访问 https://aistudio.google.com/
2. 登录你的 Google 账号
3. 创建新项目或选择已有项目
4. 在项目设置中找到 API Key
5. 复制 API Key 到 `.env` 文件

### 步骤 3: 启动服务

**方式一：使用启动脚本（推荐）**
```bash
./start.sh
```

**方式二：手动启动（需要两个终端）**

终端 1 - 启动后端：
```bash
cd server
npm run dev
```
看到 `🚀 Server is running on http://localhost:3001` 表示后端启动成功

终端 2 - 启动前端：
```bash
npm run dev
```
看到 `Local: http://localhost:3000` 表示前端启动成功

### 步骤 4: 访问应用

打开浏览器访问：**http://localhost:3000**

## 🎯 功能体验流程

### 1. 生成新选题

1. 点击左侧边栏的 **"开始生成新选题"** 按钮
2. 选择目标性别（通用/男士/女士）
3. （可选）选择面部特征和生活场景
4. 选择或输入你的视觉困扰（如"脸大显胖"、"颅顶扁平"等）
5. 在左侧选择：
   - **结构帽型库**：选择 1-3 个帽型
   - **修饰逻辑参考**：选择 1-5 个修饰逻辑点
6. 点击 **"生成深度方案并进入看板"** 按钮
7. 等待 AI 生成（通常需要 10-30 秒）

### 2. 查看内容资产看板

生成完成后会自动跳转到看板页面，你可以：
- 查看生成的选题标题和标签
- 浏览所有内容资产卡片（默认 12 张）
- 每张卡片包含：
  - 主文案 (Main Copy)
  - 视觉策略 (Visual Strategy)
  - 结构细节 (Structural Detail)
  - 美学目标 (Aesthetic Goal)

### 3. 调整卡片数量

在项目详情页面的右上角，可以点击数字按钮（10/12/15/18/20）来重新生成不同数量的卡片。

### 4. 管理项目

- **查看历史记录**：左侧边栏显示所有历史项目
- **编辑标题**：点击项目标题可以编辑
- **保存方案**：点击右上角"保存方案"按钮
- **删除项目**：鼠标悬停在项目卡片上，点击删除图标

## 🔍 功能特点

## ⚠️ 常见问题

### 后端启动失败
- 检查 `server/.env` 文件是否存在
- 确认 `GEMINI_API_KEY` 已正确配置
- 检查端口 3001 是否被占用

### 前端无法连接后端
- 确认后端服务器正在运行（检查终端1）
- 打开浏览器开发者工具，查看 Network 标签页的错误信息
- 确认后端地址是 `http://localhost:3001`

### API 调用失败
- 检查 Gemini API Key 是否有效
- 确认 API Key 对应的项目已开启账单
- 查看后端终端日志中的错误信息

### 生成速度慢
- 这是正常的，AI 生成需要时间（10-30秒）
- 如果超过 1 分钟，可能是 API Key 问题或网络问题

## 📝 提示

- 可以同时生成多个项目，所有项目都会保存在左侧边栏
- 建议先选择 1-2 个帽型和 2-3 个修饰逻辑点，效果更好
- 自定义痛点输入更具体的问题，生成的内容会更精准
