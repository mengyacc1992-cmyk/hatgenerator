# 🚀 如何启动前端和后端服务

## 问题说明

从你的截图看到：
- ✅ 后端服务正在运行（端口 3001）
- ❌ 前端服务无法连接（端口 3000）

**原因：前端服务没有运行**

## 解决方法

### 需要两个终端窗口

这个应用需要**两个服务同时运行**：
1. **后端服务**（端口 3001）- 已经在运行 ✅
2. **前端服务**（端口 3000）- 需要启动 ❌

### 步骤 1：保持后端窗口打开

**不要关闭**显示这些信息的终端窗口：
```
🚀 Server is running on http://localhost:3001
📡 CORS enabled for origin: http://localhost:3000
🔑 API Key configured: Yes
```

### 步骤 2：启动前端服务

1. **打开一个新的终端窗口**
   - 按 `Cmd + 空格`，输入"终端"，按回车
   - 或者：应用程序 → 实用工具 → 终端

2. **运行这些命令**（每行按回车）：
   ```bash
   cd /Users/chengchi/yadamengisgenius-1/hatgenerator
   npm run dev
   ```

3. **看到这样的信息就成功了：**
   ```
   VITE v6.x.x  ready in xxx ms
   ➜  Local:   http://localhost:3000/
   ```

4. **保持这个窗口也打开**（不要关闭）

### 步骤 3：刷新浏览器

在浏览器中：
- 按 `F5` 或 `Cmd + R` 刷新页面
- 或者关闭标签页，重新打开 `http://localhost:3000`

## 成功标志

现在你应该有**两个终端窗口**：

**窗口 1（后端）：**
```
🚀 Server is running on http://localhost:3001
📡 CORS enabled for origin: http://localhost:3000
🔑 API Key configured: Yes
```

**窗口 2（前端）：**
```
VITE v6.x.x  ready in xxx ms
➜  Local:   http://localhost:3000/
```

**浏览器：**
- 能正常显示应用界面（不是错误页面）

## 重要提示

⚠️ **两个终端窗口都必须一直开着！**

- 关闭后端窗口 → 无法生成内容
- 关闭前端窗口 → 网站打不开
- 就像电视要开着才能看，关了就看不了

## 如果还是不行

1. **检查两个窗口是否都在运行**
2. **确认浏览器地址是** `http://localhost:3000`（注意是 localhost，不是 localhots）
3. **尝试强制刷新**：按 `Cmd + Shift + R`（Mac）

## 一键启动（以后用）

如果觉得开两个窗口麻烦，可以用启动脚本：

```bash
./start.sh
```

这会自动启动两个服务。
