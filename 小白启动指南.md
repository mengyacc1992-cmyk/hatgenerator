# 🎯 小白启动指南（超简单版）

## 问题说明

你看到的错误页面意思是：**网站还没启动，所以打不开**。

就像你要看电视，但电视还没开机一样。我们需要先"开机"（启动服务）。

## 解决方案（两种方式）

### 方式一：最简单（推荐）✨

**只需要一个命令：**

1. 打开终端（Terminal）
2. 复制粘贴这个命令，按回车：
   ```bash
   ./start.sh
   ```

3. 等待看到类似这样的提示：
   ```
   ✅ 服务已启动!
      前端: http://localhost:3000
      后端: http://localhost:3001
   ```

4. 然后打开浏览器，访问：**http://localhost:3000**

**就这么简单！** 🎉

---

### 方式二：手动启动（如果方式一不行）

需要打开**两个终端窗口**（就像开两个窗口一样）：

#### 第一个窗口（后端服务器）

1. 打开终端
2. 输入这些命令，每行按回车：
   ```bash
   cd /Users/chengchi/yadamengisgenius-1/hatgenerator/server
   npm run dev
   ```
3. 看到 `🚀 Server is running on http://localhost:3001` 就成功了
4. **这个窗口不要关闭！** 让它一直运行

#### 第二个窗口（前端网站）

1. **新开一个终端窗口**（不要关闭第一个）
2. 输入这些命令，每行按回车：
   ```bash
   cd /Users/chengchi/yadamengisgenius-1/hatgenerator
   npm run dev
   ```
3. 看到 `Local: http://localhost:3000` 就成功了
4. **这个窗口也不要关闭！**

#### 最后

两个窗口都运行后，打开浏览器访问：**http://localhost:3000**

## 常见问题

### Q: 什么是终端？
A: 终端就是那个黑底白字的窗口，用来输入命令的。在 Mac 上按 `Cmd + 空格`，输入"终端"就能找到。

### Q: 命令是什么？
A: 命令就是你要告诉电脑做什么的"指令"。比如 `npm run dev` 就是告诉电脑"启动开发服务器"。

### Q: 为什么要两个窗口？
A: 这个应用需要两个"程序"同时运行：
- 后端：处理数据（像后台工作人员）
- 前端：显示网页（像前台接待）

### Q: 窗口可以关闭吗？
A: **不能！** 这两个窗口必须一直开着，关闭了网站就打不开了。就像电视开着才能看，关了就看不了。

### Q: 怎么停止服务？
A: 在终端窗口按 `Ctrl + C`（Mac 上可能需要按两次）

### Q: 还是打不开怎么办？
A: 
1. 检查两个窗口是否都在运行
2. 确认浏览器地址是 `http://localhost:3000`（注意是 localhost，不是 localhots）
3. 看看终端窗口有没有红色错误信息
