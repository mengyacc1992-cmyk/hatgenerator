**当前状态：** API Key 还未设置（仍是示例值 `your_gemini_api_key_here`）

**如何配置：**

1. **编辑环境变量文件**
   ```bash
   # 使用你喜欢的编辑器
   nano server/.env
   # 或
   code server/.env
   # 或
   vim server/.env
   ```

2. **找到这一行并替换**
   ```bash
   # 将这行：
   GEMINI_API_KEY=your_gemini_api_key_here
   
   # 替换为：
   GEMINI_API_KEY=你的实际_api_key_在这里
   ```

3. **获取 Gemini API Key**
   - 访问：https://aistudio.google.com/
   - 使用 Google 账号登录
   - 创建新项目或选择已有项目
   - 在项目设置中找到 **API Key**
   - 复制 API Key 到 `.env` 文件

## ⚠️ 常见问题

**Q: API Key 格式是什么样的？**
A: Gemini API Key 通常以 `AIza` 开头，长度约 39 个字符

**Q: 如何验证 API Key 是否有效？**
A: 启动后端服务，如果 API Key 无效，会在控制台显示错误信息

**Q: API Key 需要开启账单吗？**
A: 是的，Gemini API 需要项目开启账单才能使用
